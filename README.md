# TEMP_HUMD_DHT22
ESP32 is attached with DHT22 sensor, which gives currently temperature and humidity values.

**Pinout:**

- PIN#1 is attached with VCC on ESP32
- PIN#2 is attached with PIN#5 on ESP32
- PIN#3 is attached with GND on ESP32
